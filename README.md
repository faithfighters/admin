# FaithFighters 2 - Admin Frontend

This is the admin frontend application for the FaithFighters 2 platform.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) (or whichever port is configured) with your browser to see the result.
